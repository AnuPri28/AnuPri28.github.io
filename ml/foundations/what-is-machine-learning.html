What is Machine Learning?

Understanding how machines learn patterns from data and use those patterns to make predictions or decisions.

2. The Main Idea

The main idea of Machine Learning is that a computer system can learn useful patterns from data.

The model receives information as input, learns relationships from examples, and produces an output.

The key idea

A machine learning model learns a relationship between input data and an expected output.

3. Important Definitions

Machine Learning

Data

Data is the information from which a machine learning system can learn patterns.

Model

A model is a mathematical representation that learns a relationship from data.

Prediction

A prediction is the output produced by the model after processing input data.

4. Mathematical View

A simple way to represent a machine learning system is:

y = f(x)

Here:

  • x represents the input.
  • f represents the function or model.
  • y represents the output.

In machine learning, the function is usually not manually written as a fixed set of rules. Instead, the model learns parameters from data.

ŷ = f(x; θ)

Here, θ represents the parameters learned by the model.

5. Intuition

Imagine teaching a child to recognize cats and dogs.

Instead of giving the child a complete list of rules about ears, eyes, fur, and body shape, we can show many examples.

From these examples, the child gradually learns patterns that help distinguish a cat from a dog.

💡 The machine learning process is similar: the system receives examples and learns patterns that can later be used to make predictions.

6. Example

Suppose we want to predict the price of a house.

The input data might include:

  • House size
  • Number of bedrooms
  • Location
  • Age of the house

The target output is the house price.

Input → Model → Output

House information → Machine Learning Model → Predicted Price
